### 這是什麼？
- MPlus是一個基於Python的Markdown實時編輯器，支持 P(Preview) E(Edit) R(Render) 三大功能
- MPlus追求快速啓動，快速編輯，即時渲染，以此方便用戶快速創建/編輯Markdown文件

### 它的功能被閹割了嗎？
- 事實上，我們只做了精簡化處理，並未閹割Markdown基礎標記符號。
- 您可以使用任意Markdown標記符號對您的文本文件進行處理！
